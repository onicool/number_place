# Number Place

Safari と Chrome でそのまま動く、子供向けの Number Place ゲームです。

## 使い方

- `index.html` を Safari または Chrome で開く
- iPad でも PC でもプレイ可能
- セーブデータはブラウザの `localStorage` に保存される

## ファイル構成

- `index.html`: 画面構造
- `styles.css`: レイアウトと見た目
- `app.js`: ゲームロジック、保存、ヒント、取り消し

## 実装済み機能

- ホーム画面
- 難易度選択
- 9x9 の盤面操作
- 行・列・ブロックのハイライト
- 重複チェック表示
- ヒント
- ひとつもどす
- 途中保存とつづきから
- クリア表示
